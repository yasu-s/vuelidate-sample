# vuelidate-sample

## 概要 

- Vue3 +  Vuelidateで作成したバリデーションのサンプルです。

## 動作環境

- Node.js - 16.x
- Yarn - 1.22.x

## ライブラリ

- vue - 3.2.x
- @vuelidate/core - 2.0.0-alpha.42
- @vuelidate/validators - 2.0.0-alpha.30

## 動作確認

```bash
# パッケージインストール
yarn

# 起動
yarn dev
```
